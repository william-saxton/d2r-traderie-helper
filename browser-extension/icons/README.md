# Icons Directory

This directory should contain three PNG icon files for the browser extension:

- `icon16.png` (16x16 pixels) - Used in the browser toolbar
- `icon48.png` (48x48 pixels) - Used in extension management pages
- `icon128.png` (128x128 pixels) - Used in Chrome Web Store and high-DPI displays

## How to Create Icons

### Option 1: Use the Icon Generator (Easiest)
1. Open `create-placeholder-icons.html` in your browser
2. Click "Generate All Icons"
3. Download each icon using the buttons
4. Save them in this directory with the correct names

### Option 2: Use Online Tools
- [favicon.io](https://favicon.io/) - Generate favicons
- [Canva](https://www.canva.com) - Design custom icons
- [Figma](https://www.figma.com) - Professional design tool

### Option 3: Find D2R Icons
Search for Diablo 2 related images:
- Horadric Cube
- Runes (Ber, Jah, etc.)
- Gems
- Diablo logo
- Character class symbols

Resize to required dimensions using an image editor.

## Requirements

- Format: PNG
- Transparent background recommended
- Square dimensions
- Clear and recognizable even at 16x16
- D2R or Diablo themed (optional but recommended)
